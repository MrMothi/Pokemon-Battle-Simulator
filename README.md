# Pokemon-Battle-Simulator
Terminal based Pokémon battle game made with Java

## Features
- Status effects
- Critical hits
- Type damage 
- 8 Pokémon choices
- Health
- Turn based gameplay
- Simple enemy AI based on ranndomization
- ASCII art (From Online) 

## Programming Concepts Used
- Object orientation
- Inheritance
- Methods
- ArrayList
- Error checking 
- Javadoc commenting
